## Tic Tac Toe

A fully functioning, responsive Tic Tac Toe website built with HTML, CSS, and JavaScript.

## Features

- Two-player gameplay (`X` vs `O`)
- Turn indicator with win/draw messages
- Automatic win detection for all rows, columns, and diagonals
- Draw detection when the board is full
- Winning line highlight
- Round restart button
- Scoreboard with `X` wins, `O` wins, and draws
- Score reset button
- Responsive layout for desktop and mobile

## Project Files

- `index.html`: page structure and game UI for GitHub Pages
- `styles.css`: responsive styling, theme, and animations
- `script.js`: game state, turn handling, win logic, and scoring

## Run Locally

1. Open `index.html` in your browser.
2. Click any square to place a mark.
3. Use **Restart Round** to clear the board.
4. Use **Reset Score** to clear board and scores.

## Game Rules

1. Players take turns placing `X` and `O`.
2. First player to align 3 marks horizontally, vertically, or diagonally wins the round.
3. If all 9 squares are filled with no winner, the round is a draw.
